Project Overview

Dishcovery is a modern mobile application designed to help users discover, create, and share recipes with ease. Built with Flutter, Dishcovery leverages cloud technology, real-time data synchronization, and an intuitive user interface to provide a seamless culinary experience. Whether you are a home cook, a food enthusiast, or someone who loves to share recipes with friends and family, Dishcovery offers a comprehensive platform to manage your recipes, store images, interact with a built-in chatbot for instant assistance, and connect with others.

With features such as secure authentication, cloud-based storage, a responsive design, a helpful chatbot for user support and cooking tips, and support for multiple languages and custom fonts, Dishcovery aims to make recipe management simple, enjoyable, and accessible to everyone. This user manual will guide you through the app’s features, usage instructions, and best practices to help you get the most out of your Dishcovery experience.

Key Features

Recipe Management

Create, edit, and delete recipes with detailed instructions.

User Authentication

Secure sign-in and registration using email/password or Google accounts.

Real-Time Data Synchronization

Recipes and user data are synced in real-time using cloud services.
